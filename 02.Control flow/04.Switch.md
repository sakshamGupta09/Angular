# Switch

## Why

- We have a variable or expression.
- It can have multiple possible values.
- For each value, we have to execute some different logic.

## Syntax

@switch (condition) {  
  @case (caseA) {  
    Case A.  
  }  
  @case (caseB) {  
    Case B.  
  }  
  @default {  
    Default case.  
  }  
}
